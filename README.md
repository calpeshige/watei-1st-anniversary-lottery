# 🎌 ガラポン抽選会アプリ

昭和レトロな居酒屋イベント向けのガラポン抽選アプリです。iPadでの使用を想定しています。

## 特徴

- 🎰 **リアルなガラポンアニメーション** - 5秒間の回転演出
- 🎨 **和風レトロデザイン** - 昭和の居酒屋をイメージした温かみのある配色
- 🎵 **効果音** - 回転音と当選音をWeb Audio APIで生成
- 🎉 **紙吹雪演出** - 当選時の華やかなアニメーション
- 💾 **データ永続化** - ブラウザを更新しても在庫情報を保持
- ⚙️ **カスタマイズ可能** - 景品名・確率・在庫を自由に編集
- 🎨 **色の追加** - 最大11色まで景品を追加可能
- 📱 **iPad最適化** - タッチ操作とフルスクリーン表示に対応

## 初期設定

### 景品と確率
- 金（特賞）: 1個 - 3%
- 銀（1等）: 2個 - 7%
- 銅（2等）: 3個 - 10%
- 赤（3等）: 150個 - 70%
- 緑（参加賞）: 100個 - 10%

## 使い方

### 1. 開発環境で実行

```bash
# 依存関係をインストール
npm install

# 開発サーバーを起動
npm run dev
```

ブラウザで [http://localhost:5173](http://localhost:5173) を開きます。

### 2. 本番ビルド

```bash
# ビルド
npm run build

# プレビュー
npm run preview
```

### 3. Vercelにデプロイ

```bash
# Vercel CLIをインストール（初回のみ）
npm i -g vercel

# デプロイ
vercel
```

または、GitHubリポジトリに push して Vercel の Web UI から自動デプロイ。

## 機能説明

### メイン画面
- **回すボタン**: ガラポンを回して抽選を開始
- **景品一覧**: 現在の在庫と確率を表示
- **残り数**: 全景品の合計在庫を表示

### 景品管理画面
- **景品名の編集**: 各色の景品名を変更
- **在庫管理**: 在庫数を増減
- **確率調整**: 各色の当選確率を変更（合計100%推奨）
- **色の追加**: 新しい色を自動で追加
- **削除**: 不要な色を削除（最低1色は必要）

### リセット機能
- 全ての在庫を初期値に戻す
- 確認ダイアログで誤操作を防止

## 技術スタック

- **フレームワーク**: React 18 + TypeScript
- **ビルドツール**: Vite
- **スタイリング**: Tailwind CSS v3
- **アニメーション**: Framer Motion
- **紙吹雪**: canvas-confetti
- **効果音**: Web Audio API
- **データ永続化**: LocalStorage
- **デプロイ**: Vercel

## iPad での使用

### 推奨設定
1. Safari でアプリを開く
2. 共有ボタン → 「ホーム画面に追加」
3. フルスクリーンアプリとして起動
4. 横向きでの使用を推奨

### タッチ操作
- すべてのボタンがタッチに最適化
- スワイプ操作は不要
- テキスト選択は入力フィールドのみ有効

## カスタマイズ

### 色の追加順序
金 → 銀 → 銅 → 赤 → 緑 → 紫 → 青 → オレンジ → ピンク → 水色 → 黄色

### 効果音のカスタマイズ
`src/utils/sounds.ts` で音の周波数や長さを調整可能

### デザインのカスタマイズ
`tailwind.config.js` で配色を変更
`src/index.css` でカスタムスタイルを追加
